# ❄ Snow Travels Website

Northern lights tour website for Snow Travels, Tromsø.

## Files
- `index.html` — main page
- `styles.css` — all design and layout
- `script.js` — language toggle, AI chat, navigation
- `images/` — put your photos here (see below)

## Add your photos
Create a folder called `images/` and add:
| Filename | What it shows |
|---|---|
| `vip-tour.jpg` | Your best aurora/VIP tour photo |
| `bus-tour.jpg` | Group bus tour or aurora photo |
| `hassan.jpg` | Hassan's guide photo |
| `kamil.jpg` | Kamil's guide photo |
| `marius.jpg` | Marius's guide photo |

Photos will automatically appear. If a photo is missing, a nice emoji placeholder shows instead.

## Change prices or text
Open `index.html` and search for the text you want to change. Everything is clearly labelled.

## Deploy to Vercel
1. Go to vercel.com
2. Connect your GitHub account
3. Import `hugosins6969/snow-travels-website`
4. Click Deploy — done! Your site is live.

## Point your domain
In Vercel → Settings → Domains → add `snow-travels.com`
Then in your domain registrar (where you bought the domain), point DNS to Vercel.

## Update the site anytime
Just edit any file on GitHub → the site updates automatically within 30 seconds.
